# 鋒兄AI資訊系統 🚀

一個基於 Electron 的桌面應用程式，整合了多媒體管理、CRUD 功能和多語言音樂播放系統。

## ✨ 主要功能

### 🏠 首頁儀表板
- 系統概覽和統計資訊
- 技術堆疊展示（Electron + Contentful CMS）
- 快速導航到各功能模組

### 🖼️ 圖片庫管理
- 自動掃描 `assets/images` 資料夾（244+ 張圖片）
- 支援 JPG, PNG, GIF, WebP 格式
- 圖片預覽和檢視功能
- 智能載入和分頁顯示

### 🎬 影片庫管理
- 自動掃描 `assets/videos` 資料夾
- 支援 MP4, AVI, MOV, WMV, MKV 格式
- 內建影片播放器
- 影片資訊和元數據顯示

### 🎵 鋒兄音樂庫（多語言支援）
- **史上最瞎結婚理由** - 中英日三語版本
- **鋒兄進化Show🔥** - 中英日三語版本
- 實時音樂播放功能
- 多語言歌詞檢視和下載
- 音樂檔案統計和管理

### 🍎 食品管理系統（CRUD）
- ✅ **Create**: 新增食品資訊
- 📖 **Read**: 瀏覽所有食品列表
- ✏️ **Update**: 編輯食品詳情
- 🗑️ **Delete**: 刪除食品記錄
- 📊 到期日提醒和統計分析
- 🔍 智能搜尋功能

### 📊 訂閱管理系統（CRUD）
- ✅ **Create**: 新增訂閱服務
- 📖 **Read**: 管理所有訂閱
- ✏️ **Update**: 更新訂閱資訊
- 🗑️ **Delete**: 取消訂閱
- ⚠️ 付款到期提醒
- 📈 訂閱統計儀表板

### 🔧 系統儀表板
- 即時數據統計
- 到期提醒系統
- 系統健康監控

## 🛠️ 技術堆疊

### 前端技術
- **Electron** - 桌面應用程式框架
- **HTML5 + CSS3 + JavaScript** - 現代化 Web 技術
- **響應式設計** - 支援多種螢幕尺寸

### 後端技術
- **Contentful CMS** - 雲端內容管理系統
- **Node.js** - 檔案系統操作
- **RESTful API** - 數據交互

### 核心功能
- **多語言支援** - 中文、English、日本語
- **音樂播放** - 內建 HTML5 Audio 播放器
- **檔案管理** - 自動掃描和索引
- **CRUD 操作** - 完整的數據管理
- **即時搜尋** - 智能過濾和排序

## 📁 專案結構

```
electronkiro20251222/
├── main.js                 # Electron 主程序
├── package.json            # 專案配置
├── assets/                 # 資源檔案
│   ├── images/            # 圖片庫（244+ 張圖片）
│   ├── videos/            # 影片庫
│   └── musics/            # 音樂庫（中英日三語）
├── src/
│   ├── index.html         # 主頁面
│   ├── js/                # JavaScript 模組
│   │   ├── app.js         # 主應用程式邏輯
│   │   ├── songManager.js # 音樂管理（多語言）
│   │   ├── imageManager.js# 圖片管理
│   │   ├── videoManager.js# 影片管理
│   │   ├── crudManager.js # CRUD 功能
│   │   └── contentfulManager.js # CMS 整合
│   └── styles/            # CSS 樣式
│       ├── main.css       # 主要樣式
│       └── components.css # 組件樣式
```

## 🚀 安裝和運行

### 環境需求
- Node.js 16+ 
- npm 或 yarn

### 安裝步驟

1. **克隆專案**
```bash
git clone https://github.com/goldshoot0720/electronkiro20251222.git
cd electronkiro20251222
```

2. **安裝依賴**
```bash
npm install
```

3. **啟動應用程式**
```bash
npm start
```

### 開發模式
```bash
npm start -- --dev  # 開啟開發者工具
```
