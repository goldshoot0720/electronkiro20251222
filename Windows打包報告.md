## 打包過程

### 遇到的問題和解決方案

#### 1. electron-builder 代碼簽名問題
**問題**: electron-builder 在嘗試下載和解壓代碼簽名工具時失敗
```
ERROR: Cannot create symbolic link : 系統管理員權限不足
```

**解決方案**: 改用 electron-packager，避免複雜的代碼簽名流程

#### 2. 圖標格式問題
**問題**: ICO 圖標格式不正確導致打包失敗
```
Reserved header is not 0 or image type is not icon
```

**解決方案**: 暫時移除自定義圖標，使用 Electron 默認圖標

#### 3. 中文檔名支援
**問題**: Windows 系統對中文檔名的支援
**解決方案**: 成功支援中文應用程式名稱和執行檔名稱
