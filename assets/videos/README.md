## 支援的格式
- MP4
- AVI
- MOV
- WMV
- MKV

## 使用方式
1. 將影片檔案放入此資料夾
2. 系統會自動掃描並顯示在影片庫中
3. 支援拖放上傳功能

## 注意事項
- 建議影片大小不超過 100MB
- 檔案名稱請使用英文或數字
- 避免使用特殊字元
